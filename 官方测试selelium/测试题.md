以下是围绕 Selenium 官方测试页面 `https://www.selenium.dev/selenium/web/web-form.html` 设计的练习题，涵盖**元素定位、基础交互、特殊控件操作**等核心知识点，适合 Selenium 自动化初学者练习：


### 练习一：基础元素交互
**要求**：编写脚本，完成以下操作：
1. 定位“Text input”输入框，输入文本 `“Selenium Practice”`；
2. 定位“Password”输入框，输入密码 `“123456”`；
3. 定位“Textarea”文本域，输入多行文本 `“这是第一行\n这是第二行”`；
4. 点击“Submit 提交”按钮。


### 练习二：下拉框（Select 标签）操作
**要求**：编写脚本，操作“Dropdown (select)”下拉框：
1. 定位该下拉框，使用 `Select` 类选择**第二个选项**（通过索引）；
2. 再次选择“Option 1”选项（通过可见文本）；
3. 最后选择 value 为 `“third”` 的选项（通过 value 属性）。


### 练习三：复选框与单选框操作
**要求**：编写脚本，完成以下操作：
1. 取消“Checked checkbox”的勾选状态；
2. 勾选“Default checkbox”；
3. 切换“Checked radio”为“Default radio”。


### 练习四：特殊控件操作
**要求**：尝试操作以下特殊控件（选做，部分控件 Selenium 原生支持有限，可结合 JavaScript 或第三方库拓展）：
1. 定位“Color picker”颜色选择器，设置颜色为 `“#ff0000”`（红色）；
2. 定位“Date picker”日期选择器，选择日期为 `“2025-11-04”`；
3. 定位“Example range”滑块，将其拖到最右侧。


### 练习五：综合场景
**要求**：编写一个完整脚本，按顺序完成以下操作：
1. 在“Text input”中输入 `“自动化测试”`；
2. 选择“Dropdown (select)”的“Option 2”；
3. 勾选“Default checkbox”；
4. 点击“Submit 提交”按钮；
5. 等待页面跳转后，验证页面标题是否包含 `“Thanks”`。


### 提示
- 元素定位优先使用 **ID、Name、标签名** 等简单方式，若需复杂定位再用 XPath/CSS 选择器；
- 操作下拉框需导入 `from selenium.webdriver.support.ui import Select`；
- 涉及等待可使用 `time.sleep()` 或 Selenium 显式等待（`WebDriverWait`）；
- 特殊控件（如颜色选择器、滑块）若原生方法不支持，可搜索“Selenium 操作 + 控件名称”学习拓展方案（如 JavaScript 注入、ActionChains 等）。

通过这些练习，可逐步掌握 Selenium 对文本、表单、下拉框、按钮等核心元素的操作逻辑，为复杂场景自动化打下基础。
